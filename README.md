# College Department Information System

## Overview

This Java-based desktop application provides information about various college departments, including staff details and syllabi. Built using Swing for the graphical user interface and MySQL for data storage, the application offers a user-friendly way to view and manage departmental information.

## Features

- **Department Views**: 
  - Artificial Intelligence and Machine Learning (AIML)
  - Artificial Intelligence and Data Science
  - Computer Science and Engineering (CSE)
  - Electronics and Telecommunication (E_TC)
  - Chemical Engineering

- **Data Display**:
  - Staff Details
  - Department Syllabus

- **Interactive Buttons**:
  - Toggle between viewing staff details and syllabi for each department.

## Getting Started

- Java Development Kit (JDK) 8 or later
- MySQL Database
- JDBC Driver for MySQL

   ```

2. **Setup Database**:
   - Import the provided SQL schema into your MySQL database.

3. **Compile and Run**:

   - Compile and run the application using your preferred IDE or command line tools.

### Usage

- Launch the application to access the GUI.
- Select a department to view either staff details or the syllabus.
